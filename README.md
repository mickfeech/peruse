# Peruse

Peruse is intended to give you an interface to the Plex Media Server and Plex.tv API for Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'peruse'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install peruse
